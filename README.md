# Sistema de Gestión de Biblioteca

Este es un monorepo para un sistema de gestión de biblioteca, implementado con Arquitectura Limpia, TDD, TypeScript y Node.js.

## Workspaces

-   `/domain`: Contiene la lógica de negocio pura y las entidades del dominio.
-   `/apps/backend`: Contiene la implementación del servidor, APIs y puntos de entrada.
-   `/apps/frontend`: Contendrá la interfaz de usuario.
